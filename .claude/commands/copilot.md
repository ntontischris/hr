# /copilot — Get Optimal Workflow & Prompts

Ask @copilot to design the best workflow for what you want to build.

## Usage
```
/project:copilot [describe what you want to build or do]
```

## Examples
```
/project:copilot θέλω να φτιάξω subscription page με Stripe
/project:copilot μετατροπή n8n workflow σε Python
/project:copilot προσθήκη OAuth Google login
/project:copilot το build σπάει με type errors
/project:copilot θέλω dark mode toggle
```

## Process
Delegate to @copilot agent. It will:
1. Analyze what layers the task touches
2. Write the exact sequence of commands and prompts
3. Warn about common pitfalls
4. Tell you when to /clear

## Output
Ready-to-paste prompts for Claude Code, in the right order, with the right agents and skills referenced.
